Description

An edit of axHUD using some panels from uwui and collyhud. Put a lot of work into this bad boy, hope you dudes like it. Minmode toggle for 9v9 and 12v12, with options for 12v12-6v6, and 9v9-6v6. All images are from tr_walkway.
