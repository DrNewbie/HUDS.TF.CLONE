Description

Small, centered, and simple.

I consider this the finished version of sAven's hud. Special attention was made to ensure usability on 4:3 screens, but there shouldn't be any issues for people running higher resolutions.

If any problems should arise, leave a comment here: https://steamcommunity.com/id/sAvenisgrey
